# Tarot Andino

Tarot interactivo de 78 cartas con iconografía inca y voz de amauta.

**[→ Abrir el tarot](./index.html)** · **[Galería completa](./gallery.html)**

## Las 78 cartas

- **22 Arcanos Mayores** con personificaciones andinas (Amauta, Pachamama, Sapa Inca, Mama Quilla, Inti, Tawantinsuyu...)
- **56 Arcanos Menores** en cuatro palos, cada uno con su propia región:
  - **Chakitajllas** (Bastos · fuego) — terrazas del Valle Sagrado
  - **Keros** (Copas · agua) — lago Titicaca
  - **Tumis** (Espadas · aire) — cumbres glaciares
  - **Chakanas** (Oros · tierra) — ruinas megalíticas
- **Figuras**: Chasqui (Sota), Auqui (Caballero), Coya (Reina), Sapa Inca (Rey)

## Estilo

Acuarela tradicional con sangrados de pigmento y papel visible. Textos del amauta en castellano neutro: mensaje arcano, significado y pregunta para meditación.

## Cómo usar

1. Elige **Una carta** o **Tres cartas** (Pasado · Presente · Porvenir)
2. Toca el mazo para detener el barajeo
3. Toca cada carta para revelarla
4. Lee la lectura
